# ChiyoS.Draw

如果元数据没错，这个项目应该是 2021 年 7 月 4 日启动的，这应该算是我第一个完整的项目，一个起初挺简约的、基于`C#`+`WPF`抽签程序

这个项目始于一次班级活动需要搞活动抽签，我便承担了班长的任务搞了这么一个项目

## 名称的由来

初三在追番剧`月刊少女野崎君`，`Chiyo`源自女主 佐倉千代（さくら　ちよ）中 `千代` 的罗马音.`S`可以理解为`School`，当时还突然心血来潮搞了个`H(Home)`，结果技术不足放弃了.如果硬要二次元理解的话也可以认作是`Sakura`，即`佐倉`

`Draw`便是抽签的意思（当时百度翻译出来的）

`Komari`是后面加的，这个项目经历过一次重构，重构版本代号为`Komari`，源自 Key 社经典作品`Little Busters!（リトルバスターズ！）`女主之一`神北小毬`的`小毬`的罗马音

至于为什么使用了`.`，因为`ChiyoS`下还有其他东西，对的，其实这是一个系列：

- `ChiyoS.SCR`是屏保软件，当时初中的一体机还是老旧的 SCT 中电数码的产品，电教处特意要求早上来了这电脑就可以不用关了，所以我就开始了私货发挥时刻，基于`C#`和`WinForm`弄了个能够轮播壁纸和展示格言、天气预报的屏保

- `ChiyoS.S`是一个远程控制用的，其实就是用了`go-cqhttp`在一体机上跑 QQ 机器人方便我回家后用 QQ 控制电脑执行一些命令，比如晚自习结束忘记关了电脑，可以远程关机防止第二天值班老师通报批评。起床这个软件负责保活和记录开机日志，保存 10 分钟内的电脑截图(当时真的是异想天开)。不过最好使的还是发送文件功能，能输入指定路径把文件读过来发班群造福同学

## 发展历程

### 初版

最初这个抽签程序挺纯粹的——练习`C#`与`WPF`、完成班长的任务.控件库选择的是[HandyControl](https://github.com/HandyOrg/HandyControl)，图标来源于 Iconfont，生成 ttf 文件使用

原理便是简单的生成随机数，不过生成随机数数组我是从 CSDN 还是博客园上复制的

不得不说 LLM 的出现确实极大改变了生活，当时我没有系统学过编程语言，只能靠搜索引擎一点点查资料，处理各种我看不懂的报错，翻译那些我看不懂的英文单词

![image.avif](/README/image.avif)

后来我又更换为栗山未来的背景，并且增添了左右滑动水平`StackPanel`的按钮，方便一体机的触控屏操作

![image2.avif](/README/image2.avif)

然后初步就是到此停止了，随即到来的是代号为`Komari`的重构

#### V2.2

巅峰时刻还是为了实现动态壁纸，我研究了很久怎么把网页塞到 WPF 中，起初是 WPF 自带的浏览器，但是在 CSS 与 JS 有些问题，所以我决定向大厂看齐——把**Chromium**塞到 WPF 项目中，没有 LLM 的时代，我搜了好久的资料，才研究出妥善的方法，通过速成 HTML，抄了渐变效果动画，引入花瓣飘落 JS，然后搜集到**和泉纱雾**的**お帰りなさい、お兄ちゃん**的语音，设置为启动语音，可谓是浓度巅峰时刻

不过很不幸的是，隔壁班也是用了这个软件，老师是英语老师，还是年级主任，虽然没直接说到我头上（因为老师不认识我），但是还是给搬运这个软件的同学不小的压力，所以这两个功能只存活了 1 个版本
